목표 시험 회차별 수강 시작 권장 시기
실내건축기사 — 최소 5개월
  • 1회(4~5월 실기) → 전년도 11월~12월 수강 시작
  • 2회(7~8월 실기) → 당해 2~3월 수강 시작
  • 3회(10~11월 실기) → 당해 5~6월 수강 시작
  • CAD·스케치업 선행 필요 시 +1~2개월 추가
실내건축산업기사·기능사 — 최소 4개월
  • 상반기 실기 목표 → 전년도 12월~당해 1월 수강 시작
  • 하반기 실기 목표 → 당해 4~5월 수강 시작
  • CAD·스케치업 선행 필요 시 +1~2개월 추가
  • 기능사는 응시 제한이 없어 빠른 도전 가능

수강 전 반드시 확인해야 할 사항

  • 기사 기준 5개월(산업기사·기능사 4개월)은 CAD·스케치업 선행 능력이 충분히 갖춰진 경우의 최소 기간입니다.
  • CAD 또는 스케치업 수준이 부족하면 선행 수업 기간이 추가로 필요합니다. (보통 1~3개월 선행)
  • 컴퓨터 작업 속도, 공간 이해력, 학습 시간 투자량에 따라 개인차가 크게 납니다.
  • 기사는 상세단면도가 추가되므로, 처음 도전하는 분은 6~7개월 준비를 기준으로 생각하는 것이 안전합니다.
  • 필기 시험에서 불합격하면 다음 회차로 전체 일정이 밀리므로, 필기도 충분히 준비해야 합니다.
  • 기출문제 풀이 단계에서 속도·완성도가 기준에 못 미치면 추가 연장 수강을 적극 권장합니다.
